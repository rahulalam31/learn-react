## Installation

Install my-project with npm

```bash
  git clone git@github.com:rahulalam31/learn-react.git
  cd learn-react
  npm install
  // for bmi calculator app
  git checkout bmicalc

  git checkout weather-app
  // for counter app
  git checkout counter-app
  git checkout bmicalc
  git checkout bmicalc
```
